# Точка верификации маппинга (Human-in-the-loop)

Предлагается модель проверки с несколькими ступенями.

| Уверенность AI |	Действие |
| --- | --- |
| > 95%	| маппинг принимается автоматически |
| 80–95% |	товаровед подтверждает или исправляет |
| <80% |	ручной маппинг |

После проверки товароведом:
- исправления сохраняются
- результаты используются для последующего обучения модели
- постепенно уменьшается объем ручной работы

Такой подход соответствует паттерну Human-in-the-loop, который рекомендуется использовать там, где автоматизация не может гарантировать достаточную точность принятия решения.
